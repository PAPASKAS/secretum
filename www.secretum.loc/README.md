# ROUTING

## It app use resource controller laravel as routing:

Verb | URI                   | Action
--- |-----------------------|----
POST | /photos               | index
POST | /photos/create        | create
POST | 	/photos              | store
POST | /photos/{photo}       | show
POST | /photos/{photo}/edit	 | edit
PUT/PATCH | /photos/{photo}  | update
DELETE | /photos/{photo}     | destroy
